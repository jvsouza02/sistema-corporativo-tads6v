## Cenário de Teste 014

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT014** |
| **Nome** | **Cancelar agendamento existente** |
| **Objetivo** | Verificar se o cancelamento altera corretamente o status do agendamento no banco de dados. |
| **Tipo** | Teste de integração |
| **Dado** | Que o cliente possua um agendamento ativo associado à sua conta. |
| **Comportamento** | Ao acessar a rota de cancelamento do agendamento. |
| **Resultados esperados** | O sistema deve atualizar o status do agendamento para cancelado, mantendo o registro no banco. |
