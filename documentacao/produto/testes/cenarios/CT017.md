## Cenário de Teste 017

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT017** |
| **Nome** | **Excluir atendimento existente** |
| **Objetivo** | Verificar se o sistema remove corretamente um atendimento do banco de dados. |
| **Tipo** | Teste de integração |
| **Dado** | Que exista um atendimento previamente registrado. |
| **Comportamento** | Ao acessar a rota de exclusão do atendimento (DELETE). |
| **Resultados esperados** | O sistema deve excluir o atendimento permanentemente do banco e não retorná-lo em consultas posteriores. |
