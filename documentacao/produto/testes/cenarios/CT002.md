| **Componente**      | **Descrição** |
|----------------------|---------------|
| **Identificador**    | CT002 |
| **Nome**             | Registrar atendimento com itens de valor zero |
| **Objetivo**         | Verificar se o sistema calcula corretamente o total quando há itens com preço igual a zero. |
| **Pré-condições**    | 1. Barbeiro autenticado no sistema.<br>2. Existência de serviços e produtos cadastrados com valor zero.<br>3. Existência de serviços e produtos com valor maior que zero. |
| **Dado**             | Que o barbeiro selecione itens gratuitos e pagos. |
| **Comportamento**    | Ao registrar um novo atendimento. |
| **Etapas do teste**  | 1. Acessar a tela de registro de atendimento.<br>2. Escolher um serviço com valor zero.<br>3. Escolher um produto com valor zero.<br>4. Escolher um serviço com valor maior que zero.<br>5. Registrar o atendimento. |
| **Resultados esperados** | O sistema deve somar apenas os itens com valor maior que zero e registrar o atendimento corretamente. |
