## Cenário de Teste 010

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT010** |
| **Nome** | **Login com senha vazia** |
| **Objetivo** | Verificar o comportamento do sistema quando o barbeiro tenta fazer login sem informar a senha. |
| **Tipo** | Teste unitário |
| **Dado** | Que o barbeiro informe o e-mail mas deixe o campo de senha vazio. |
| **Comportamento** | Ao tentar fazer login. |
| **Resultados esperados** | O sistema deve exibir uma mensagem informando que a senha é obrigatória e permanecer na tela de login. |
