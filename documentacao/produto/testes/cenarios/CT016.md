## Cenário de Teste 016

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT016** |
| **Nome** | **Bloquear registro de atendimento com valor negativo** |
| **Objetivo** | Garantir que o sistema não permita salvar atendimentos com valor total negativo. |
| **Tipo** | Teste de integração |
| **Dado** | Que o barbeiro informe um valor total negativo no registro do atendimento. |
| **Comportamento** | Ao tentar registrar o atendimento. |
| **Resultados esperados** | O sistema deve impedir o salvamento e retornar erro de validação. |
