## Cenário de Teste 003

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT003** |
| **Nome** | **Calcular valor total com casas decimais** |
| **Objetivo** | Confirmar se o sistema soma corretamente os valores quando há números com centavos. |
| **Tipo** | Teste unitário |
| **Dado** | Que o barbeiro selecione itens com valores decimais. |
| **Comportamento** | Ao registrar um atendimento com mais de um item. |
| **Resultados esperados** | O sistema deve somar corretamente os valores com centavos e mostrar o total exato, sem erro de arredondamento. |
