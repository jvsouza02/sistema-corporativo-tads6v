## Cenário de Teste CT023

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT023** |
| **Nome** | **Não permitir usar produto além do estoque disponível** |
| **Objetivo** | Garantir que o sistema bloqueie o uso excessivo de produtos. |
| **Tipo** | Teste exploratório |
| **Dado** | Um produto com quantidade menor que a solicitada. |
| **Comportamento** | Ao tentar usar o produto em um atendimento. |
| **Resultados esperados** | O sistema deve impedir a operação e avisar que não há estoque suficiente. |
