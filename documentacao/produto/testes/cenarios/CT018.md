### Cenário de Teste CT018

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT018** |
| **Nome** | **Impedir cadastro de serviço com valor negativo** |
| **Objetivo** | Garantir que o sistema não permita cadastrar um serviço com valor menor que zero. |
| **Tipo** | Teste exploratório |
| **Dado** | Que o usuário tente cadastrar um serviço com preço negativo. |
| **Comportamento** | Ao tentar salvar o serviço. |
| **Resultados esperados** | O sistema deve mostrar uma mensagem de erro e não permitir salvar o serviço. |
