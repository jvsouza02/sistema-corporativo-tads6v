## Cenário de Teste 001

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT001** |
| **Nome** | **Registrar atendimento sem itens** |
| **Objetivo** | Verificar o que acontece quando o barbeiro tenta registrar um atendimento sem escolher nenhum serviço ou produto. |
| **Tipo** | Teste unitário |
| **Dado** | Que o barbeiro não escolha nenhum serviço nem produto. |
| **Comportamento** | Ao tentar registrar um novo atendimento. |
| **Resultados esperados** | O sistema deve mostrar uma mensagem informando que é obrigatório escolher pelo menos um serviço ou produto e não deve permitir o registro. |
