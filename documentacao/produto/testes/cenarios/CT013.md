## Cenário de Teste 013

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT013** |
| **Nome** | **Bloquear agendamento em horário já ocupado** |
| **Objetivo** | Garantir que o sistema impeça dois agendamentos ativos no mesmo horário para a mesma barbearia. |
| **Tipo** | Teste de integração |
| **Dado** | Que já exista um agendamento com status diferente de cancelado no mesmo horário e barbearia. |
| **Comportamento** | Ao tentar registrar um novo agendamento no mesmo horário. |
| **Resultados esperados** | O sistema deve impedir o cadastro e retornar mensagem informando indisponibilidade do horário. |
