| **Componente**      | **Descrição** |
|----------------------|---------------|
| **Identificador**    | CT007 |
| **Nome**             | Login de barbeiro com e-mail incorreto |
| **Objetivo**         | Verificar o comportamento do sistema quando o barbeiro tenta entrar com e-mail não cadastrado. |
| **Pré-condições**    | 1. Sistema de login ativo.<br>2. Email não cadastrado disponível para teste. |
| **Dado**             | Que o barbeiro informe um e-mail incorreto. |
| **Comportamento**    | Ao tentar fazer login. |
| **Etapas do teste**  | 1. Acessar a tela de login.<br>2. Inserir o e-mail incorreto.<br>3. Clicar em Entrar. |
| **Resultados esperados** | O sistema deve exibir a mensagem “Usuário não encontrado” e permanecer na tela de login. |
