## Cenário de Teste 005

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT005** |
| **Nome** | **Editar atendimento e atualizar valor total** |
| **Objetivo** | Verificar se o sistema recalcula o valor total automaticamente quando o atendimento é editado. |
| **Tipo** | Teste unitário |
| **Dado** | Que o barbeiro altere os itens do atendimento. |
| **Comportamento** | Ao salvar a edição de um atendimento. |
| **Resultados esperados** | O sistema deve recalcular o valor total e mostrar o novo resultado corretamente. |
