## Cenário de Teste 015

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT015** |
| **Nome** | **Registrar atendimento com valor válido** |
| **Objetivo** | Verificar se o sistema registra corretamente um atendimento quando o valor informado é válido. |
| **Tipo** | Teste de integração |
| **Dado** | Que o barbeiro informe serviço, barbearia, barbeiro responsável e valor total maior ou igual a zero. |
| **Comportamento** | Ao enviar a requisição de registro de atendimento (POST). |
| **Resultados esperados** | O sistema deve salvar o atendimento no banco de dados e retornar mensagem de sucesso. |
