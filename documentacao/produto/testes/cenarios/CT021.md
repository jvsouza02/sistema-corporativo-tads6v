## Cenário de Teste CT021

| **Componente** | **Descrição** |
|----------------|---------------|
| **Identificador** | **CT021** |
| **Nome** | **Impedir atendimento quando produto do serviço está sem estoque** |
| **Objetivo** | Garantir que o atendimento não seja realizado se faltar produto. |
| **Tipo** | Teste exploratório |
| **Dado** | Um serviço que utiliza um produto sem quantidade disponível. |
| **Comportamento** | Ao tentar criar um atendimento usando esse serviço. |
| **Resultados esperados** | O sistema deve impedir o atendimento e informar que o produto está sem estoque. |
