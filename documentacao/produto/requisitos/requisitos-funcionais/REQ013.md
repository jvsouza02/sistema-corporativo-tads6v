# REQ013 - Visualizar Histórico de Atendimento

**ESCOPO:** Exibir atendimentos realizados.  
**PROPÓSITO:** Permitir consulta ao histórico.  
**ATOR:** Proprietário, Barbeiro  

---

### PRÉ-CONDIÇÕES
- Usuário autenticado.

### PÓS-CONDIÇÕES
- Atendimentos exibidos.

---

### FLUXO NORMAL
1. O usuário acessa o histórico de atendimentos.
2. O sistema lista os atendimentos registrados.
3. O usuário visualiza serviços, produtos, data e valor total.

---

### FLUXO DE EXCEÇÃO
- **E1 – Nenhum atendimento encontrado:** Sistema informa ausência de registros.
