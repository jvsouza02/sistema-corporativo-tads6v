# REQ005 - Registrar Atendimento

**ESCOPO:** Registrar e manter o histórico dos atendimentos realizados pelos barbeiros.  
**PROPÓSITO:** Permitir que o barbeiro registre os serviços e produtos usados em cada atendimento, com cálculo automático do valor total.  
**ATOR:** Barbeiro  

---

### PRÉ-CONDIÇÕES
- O barbeiro deve estar logado.
- Deve estar vinculado a uma barbearia.
- Devem existir serviços e produtos cadastrados.

### PÓS-CONDIÇÕES
- O atendimento é salvo com sucesso.
- O histórico da barbearia é atualizado.

---

### FLUXO NORMAL
1. O barbeiro acessa “Atendimentos”.
2. O sistema mostra a lista de atendimentos realizados.
3. O barbeiro clica em “Novo Atendimento”.
4. O sistema exibe os serviços e produtos disponíveis.
5. O barbeiro seleciona os itens e confirma o registro.
6. O sistema calcula automaticamente o valor total e salva.
7. O sistema mostra uma mensagem de sucesso.

---

### FLUXO DE EXCEÇÃO
- **E1 - Nenhum serviço selecionado:** O sistema mostra mensagem pedindo para escolher pelo menos um serviço.
- **E2 - Erro ao salvar:** O sistema mostra mensagem pedindo para tentar novamente.
