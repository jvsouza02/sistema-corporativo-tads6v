# REQ004 - Gerenciar Barbeiros

**ESCOPO:** Cadastrar, editar, remover e transferir barbeiros entre barbearias do proprietário.  
**PROPÓSITO:** Permitir que o proprietário registre barbeiros, atualize seus dados e transfira profissionais entre unidades.  
**ATOR:** Proprietário  

---

### PRÉ-CONDIÇÕES
- O proprietário deve estar logado.
- Deve existir pelo menos uma barbearia cadastrada.

### PÓS-CONDIÇÕES
- O barbeiro é cadastrado e vinculado a uma barbearia.
- O sistema atualiza a lista de barbeiros.

---

### FLUXO NORMAL
1. O proprietário acessa a área “Barbeiros”.
2. O sistema mostra a lista de barbeiros cadastrados.
3. O proprietário clica em “Novo Barbeiro”.
4. O sistema exibe o formulário de cadastro.
5. O proprietário preenche os dados e confirma.
6. O sistema valida as informações e salva o registro.
7. O sistema mostra mensagem de sucesso.

---

### FLUXO DE EXCEÇÃO
- **E1 - Campos obrigatórios vazios:** O sistema mostra mensagem pedindo o preenchimento dos campos.
- **E2 - E-mail já usado:** O sistema mostra mensagem avisando que o e-mail já está cadastrado.
- **E3 - Erro ao salvar:** O sistema mostra mensagem pedindo para tentar novamente.
