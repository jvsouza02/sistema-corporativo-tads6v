# REQ003 - Realizar Login

**ESCOPO:** Acesso ao sistema através de e-mail e senha.  
**PROPÓSITO:** Permitir que usuários acessem sua área correspondente.  
**ATOR:** Proprietário, Barbeiro, Cliente  

---

### PRÉ-CONDIÇÕES
- Possuir cadastro.
- Informar e-mail e senha válidos.

### PÓS-CONDIÇÕES
- Usuário autenticado.
- Redirecionamento conforme seu tipo:
  - Proprietário → tela inicial (lista de barbearias) 
  - Barbeiro → página de agendamentos do barbeiro
  - Cliente → lista de barbearias disponíveis

---

### FLUXO NORMAL
1. O usuário acessa a tela de login.
2. O sistema exibe campos e-mail e senha.
3. O usuário preenche as credenciais.
4. O sistema valida os dados.
5. O usuário é direcionado para sua área correspondente.

---

### FLUXO DE EXCEÇÃO
- **E1 – Credenciais inválidas:** Sistema informa erro.  
- **E2 – Conta inexistente:** Sistema informa que não existe cadastro.  
- **E3 – Erro inesperado:** Sistema informa falha.
