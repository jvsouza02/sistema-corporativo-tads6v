# REQ010 – Realizar Agendamento

**ESCOPO:** Registrar agendamentos do cliente.  
**PROPÓSITO:** Permitir que o cliente escolha barbearia, serviço, data e horário.  
**ATOR:** Cliente  

---

### PRÉ-CONDIÇÕES
- Cliente autenticado.
- Barbearia e serviços disponíveis.
- Horários dentro do expediente.

### PÓS-CONDIÇÕES
- Agendamento registrado e vinculado ao cliente.

---

### FLUXO NORMAL
1. Cliente acessa barbearias disponíveis.  
2. Sistema mostra serviços e horários livres.  
3. Cliente seleciona barbearia, serviço, data e horário.  
4. Sistema valida disponibilidade.  
5. Sistema registra o agendamento e confirma ao cliente.  

---

### FLUXO DE EXCEÇÃO
- **E1 – Horário indisponível:** Sistema informa indisponibilidade.  
- **E2 – Dados incompletos:** Sistema informa o que falta.  
- **E3 – Erro ao registrar:** Sistema pede nova tentativa.
