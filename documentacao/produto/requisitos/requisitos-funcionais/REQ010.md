# REQ010 - Realizar Agendamento

**ESCOPO:** Registro de agendamentos feitos pelo cliente.  
**PROPÓSITO:** Permitir que o cliente escolha uma barbearia, veja horários disponíveis e marque um horário para atendimento.  
**ATOR:** Cliente  

---

### PRÉ-CONDIÇÕES
- Cliente deve estar autenticado.  
- Barbearia deve estar cadastrada.  
- Deve existir pelo menos um horário disponível dentro do expediente da barbearia.  

### PÓS-CONDIÇÕES
- Agendamento é criado e vinculado ao cliente, barbeiro e barbearia.  
- Cliente pode visualizar o novo agendamento no histórico.  
- Sistema atualiza a lista de horários ocupados.  

---

### FLUXO NORMAL
1. O cliente acessa a tela de agendamentos.  
2. O sistema mostra todas as barbearias disponíveis.  
3. O cliente seleciona uma barbearia.  
4. O sistema mostra os horários disponíveis conforme o expediente da barbearia.  
5. O cliente escolhe um horário.  
6. O cliente preenche os dados do agendamento: data, horário e barbeiro desejado.  
7. O sistema valida o horário selecionado.  
8. O agendamento é salvo e vinculado ao cliente.  
9. O sistema exibe confirmação do agendamento.  

---

### FLUXO DE EXCEÇÃO
- **E1 – Horário indisponível:** Sistema informa que o horário já foi preenchido.  
- **E2 – Dados incompletos:** Sistema informa quais informações precisam ser preenchidas.  
- **E3 – Erro ao salvar agendamento:** Sistema exibe mensagem de erro e solicita nova tentativa.
