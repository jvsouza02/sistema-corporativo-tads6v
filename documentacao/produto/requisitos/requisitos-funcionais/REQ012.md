# REQ012 - Registrar Atendimento

**ESCOPO:** Registro dos atendimentos realizados pelo barbeiro.  
**PROPÓSITO:** Registrar serviços, produtos e valor final.  
**ATOR:** Barbeiro  

---

### PRÉ-CONDIÇÕES
- Barbeiro autenticado.
- Barbeiro vinculado a uma barbearia.

### PÓS-CONDIÇÕES
- Atendimento registrado.
- Estoque atualizado.

---

### FLUXO NORMAL
1. O barbeiro acessa sua página de agendamentos.
2. O sistema exibe atendimentos anteriores.
3. O barbeiro preenche formulário com: Serviços prestados, Produtos utilizados, Valor total, Comentários.
4. O sistema valida que há pelo menos um serviço.
5. O sistema calcula o valor total.
6. O sistema atualiza o estoque.
7. O atendimento é registrado.

---

### FLUXO DE EXCEÇÃO
- **E1 – Nenhum serviço informado:** Sistema informa erro.  
- **E2 – Valor negativo:** Sistema informa erro.  
- **E3 – Valor não informado:** Sistema informa erro.  
- **E4 – Atendimento inexistente:** Sistema informa erro.  
- **E5 – Erro ao salvar:** Sistema notifica falha.
