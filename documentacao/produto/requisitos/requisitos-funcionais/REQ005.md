# REQ005 - Listar Barbearias

**ESCOPO:** Exibir a listagem de todas as barbearias cadastradas do proprietário, com informações principais e dados de desempenho.  
**PROPÓSITO:** Permitir que o proprietário visualize suas barbearias cadastradas e acompanhe dados como quantidade de atendimentos, número de barbeiros e valor total gerado na semana.  
**ATOR:** Proprietário  

---

### PRÉ-CONDIÇÕES
- O proprietário deve estar logado.
- Deve existir ao menos uma barbearia cadastrada.

### PÓS-CONDIÇÕES
- O sistema mostra corretamente todas as barbearias do proprietário.

---

### FLUXO NORMAL
1. O proprietário acessa “Minhas Barbearias”.
2. O sistema exibe a listagem das barbearias cadastradas.
3. Para cada barbearia, são mostrados nome, endereço, horário, quantidade de barbeiros e desempenho semanal.

---

### FLUXO DE EXCEÇÃO
- **E1 - Falha na consulta:** O sistema mostra mensagem dizendo que ocorreu um erro ao carregar os dados.
- **E2 - Nenhuma barbearia cadastrada:** O sistema mostra mensagem dizendo que não há barbearias e oferece a opção de cadastrar uma nova.
