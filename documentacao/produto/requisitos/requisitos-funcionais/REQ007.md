# **REQ007 – Gerenciar Serviço**

**ESCOPO:** Cadastro, edição e remoção de serviços oferecidos pela barbearia.  
**PROPÓSITO:** Organizar a lista de serviços oferecidos em cada unidade.
**ATOR:** Proprietário  

---

## **PRÉ-CONDIÇÕES**
- O proprietário deve estar autenticado no sistema.  
- O proprietário deve ter ao menos uma barbearia cadastrada.  
- Deve existir um catálogo de produtos para vincular aos serviços.

---

## **PÓS-CONDIÇÕES**
- Serviço criado, atualizado ou removido da barbearia.  
- Serviço fica disponível para uso nos atendimentos realizados pelos barbeiros.  

---

### FLUXO NORMAL
1. O proprietário acessa a **página de serviços da barbearia**.
2. O sistema exibe serviços cadastrados.
3. O proprietário escolhe cadastrar, editar ou remover serviço.
4. O proprietário preenche formulário com: nome, descrição, preço, produtos associados.
5. O sistema salva.
6. O sistema exibe a lista atualizada.

---

### FLUXO DE EXCEÇÃO
- **E1 – Dados incompletos:** Sistema informa erro.  
- **E2 – Erro ao salvar:** Sistema notifica falha.  
- **E3 – Serviço usado em atendimento:** Sistema solicita confirmação.
