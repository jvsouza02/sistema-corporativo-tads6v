# REQ004 - Gerenciar Barbeiros

**ESCOPO:** Gestão de barbeiros existentes: edição de horários, transferência entre barbearias e remoção.  
**PROPÓSITO:** Permitir que proprietários administrem a equipe de barbeiros entre suas barbearias.  
**ATOR:** Proprietário  

---

### PRÉ-CONDIÇÕES
- Proprietário autenticado.
- Pelo menos uma barbearia cadastrada.
- Barbeiros previamente cadastrados no sistema.

### PÓS-CONDIÇÕES
- Horários de trabalho atualizados.
- Barbeiro transferido entre barbearias do mesmo proprietário.
- Barbeiro removido do sistema.
- Lista de barbeiros atualizada em tempo real.

---

### FLUXO NORMAL
1. Proprietário acessa a lista de barbeiros de uma barbearia específica.
2. O sistema mostra barbeiros cadastrados na barbearia selecionada.
3. O sistema lista outras barbearias do proprietário para transferência.
4. O proprietário pode:
   - Editar horários;
   - Transferir barbeiros;
   - Remover barbeiros.
5. O sistema exibe confirmação para cada ação realizada.

---

### FLUXO DE EXCEÇÃO
- **E1 – Barbearia não existe:** Sistema redireciona para o dashboard com aviso.  
- **E2 – Erro ao alterar dados:** Sistema informa falha na edição.  
- **E3 – Erro ao transferir:** Sistema informa falha na transferência.  
- **E4 – Erro ao remover:** Sistema informa falha na remoção.
