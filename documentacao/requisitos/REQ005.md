# REQ005 — Gerar Relatórios

**ESCOPO:** Permitir que o proprietário acesse relatórios de desempenho, exibindo informações consolidadas sobre o faturamento e o volume de atendimentos de cada barbearia, com filtragem dos dados por períodos semanais e mensais.  
**PROPÓSITO:** Fornecer ao proprietário uma visão clara do desempenho financeiro e operacional de cada unidade, permitindo o acompanhamento da performance do negócio.  
**ATOR:** Proprietário  

---

## PRÉ-CONDIÇÕES
- O proprietário deve estar autenticado no sistema.  
- Devem existir atendimentos registrados com status **“Concluído”** para geração dos dados.  

---

## PÓS-CONDIÇÕES
- O sistema exibe o relatório consolidado com os dados de faturamento e volume de atendimentos para o período e barbearia(s) selecionados.  

---

## FLUXO NORMAL
1. O proprietário acessa o módulo **“Relatórios”**.  
2. O sistema exibe a interface para seleção dos filtros do relatório.  
3. O proprietário seleciona a(s) barbearia(s) desejada(s) e o período (**semanal** ou **mensal**).  
4. O sistema valida as seleções.  
5. O sistema busca e consolida as informações de faturamento e volume, considerando apenas os atendimentos com status **“Concluído”**.  
6. O sistema exibe o relatório de desempenho com o faturamento acumulado e a quantidade de atendimentos para o período e barbearia(s) selecionados.  

---

## FLUXO DE EXCEÇÃO
**E1 — Seleção de período ausente:**  
O sistema exibe mensagem de erro informando que o período (**semanal** ou **mensal**) deve ser selecionado.  

**E2 — Nenhum dado encontrado:**  
Caso não existam atendimentos **“Concluídos”** para o período selecionado, o sistema informa que não há dados para exibir no relatório.  

**E3 — Falha no processamento:**  
O sistema informa o erro e orienta o usuário a tentar novamente.  

---

## FLUXO ALTERNATIVO
**A1 — Mudar visualização:**  
1. O proprietário pode mudar a forma como o relatório é mostrado.  
2. O sistema atualiza a exibição com os mesmos dados.  

**A2 — Comparar períodos:**  
1. O proprietário pode escolher dois períodos diferentes para comparar os resultados.  
2. O sistema mostra as diferenças entre os períodos selecionados.  

**A3 — Ver detalhes:**  
1. O proprietário pode clicar em uma barbearia ou período específico.  
2. O sistema mostra as informações detalhadas dessa seleção.
