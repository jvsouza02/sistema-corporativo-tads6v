| **Componente**      | **Descrição** |
|----------------------|---------------|
| **Identificador**    | CT005 |
| **Nome**             | Impedir valor negativo no atendimento |
| **Objetivo**         | Garantir que o sistema não aceite itens com valor negativo. |
| **Pré-condições**    | 1. Atendimento disponível para edição.<br>2. Barbeiro autenticado.<br>3. Catálogo contendo item com valor negativo (para teste). |
| **Dado**             | Que o barbeiro tente incluir um item com valor negativo. |
| **Comportamento**    | Ao salvar o atendimento com itens alterados. |
| **Etapas do teste**  | 1. Acessar a lista de atendimentos.<br>2. Escolher um atendimento para editar.<br>3. Tentar adicionar um item com valor negativo.<br>4. Salvar o atendimento. |
| **Resultados esperados** | O sistema deve mostrar uma mensagem de erro e não permitir salvar o atendimento com valores negativos. |
