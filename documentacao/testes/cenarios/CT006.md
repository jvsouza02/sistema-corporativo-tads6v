| **Componente**      | **Descrição** |
|----------------------|---------------|
| **Identificador**    | CT006 |
| **Nome**             | Login de barbeiro com e-mail correto |
| **Objetivo**         | Verificar se o barbeiro consegue acessar o sistema com e-mail válido. |
| **Pré-condições**    | 1. Barbeiro cadastrado no sistema.<br>2. Email válido registrado.<br>3. Sistema de login funcionando. |
| **Dado**             | Que o barbeiro informe um e-mail válido. |
| **Comportamento**    | Ao realizar o login. |
| **Etapas do teste**  | 1. Acessar a tela de login.<br>2. Inserir o e-mail válido.<br>3. Clicar em Entrar. |
| **Resultados esperados** | O sistema deve autenticar o barbeiro e abrir a tela inicial com os atendimentos. |
